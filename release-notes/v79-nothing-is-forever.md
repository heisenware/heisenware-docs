## Features

* First version of PDF Templating
* Entire platform is available as on-premise, single-container deployment suitable for edge hardware like IPCs of either x86 or arm64 architecture
* Tenancy selection for input widgets (inputs can be explicitly shared)
* Internal mailing service
* Support for many-to-many associations between tables
* Support for converting non-history tables to history tables including tracking of all associations
* Busylight driver available as edge-connector
* Thousands indicators are available on number input

* Tag Box in form input is now working
